Search

Search through one or more files for rows of data that match your criteria creating a new file.


Search through one or more files for rows of data that match your criteria creating a new file.

You can upload a file containing a column of values that you want to search for in the other files.

You can also use conditions to describe exactly what it is that you want the algorithm to search for.

Examples:

Search for a list of names in multiple files to find all the sales of a certain group of employees.


Find all the the sales invoices that start with the number '00051'.


Extract all rows from each file where START_DATE is between March 1st and July 31st and STATUS is 'Active'.

Combine

Join files together by finding common values and adding columns from other datasets. Select one column from each uploaded file. The values in these columns will be compared. If the values are equal, the columns from one dataset will be added to the other where the values match. This will add more data to your dataset by adding the relevant columns for the matching rows.

Calculate

Describe what you want to calculate and a column will be added to your dataset with the result. The algorithm will generate a formula based on what you need and that formula will be applied to your dataset. The result will be a new column containing a formula which calculates what you need.

Compare

Compare two datasets to find which values match and don't match.
